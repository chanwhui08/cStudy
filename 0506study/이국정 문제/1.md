문제 2
문제 제목
숫자 압축 출력 프로그램
문제 설명

정수 7개를 입력받아 배열에 저장한다.
연속해서 같은 숫자가 나오면 숫자와 반복 횟수를 압축해서 출력하시오.

예를 들어:

1 1 1 2 2 3 1

출력:

1(3) 2(2) 3(1) 1(1)

여기서 1(3)은 1이 연속으로 3번 나왔다는 뜻이다.

주의할 점은 떨어져 있는 같은 숫자는 하나로 합치지 않는다.

1 1 2 1

이 경우는:

1(2) 2(1) 1(1)

이다.

입출력 화면
정수 7개 입력: 1 1 1 2 2 3 1
압축 결과: 1(3) 2(2) 3(1) 1(1)
필요기술 및 알고리즘
배열
반복문
현재 값 저장
연속 개수 세기
구간 종료 판단
(선정이유 : 그냥 이런 유형문제 처음봐서)

문제 3
문제 제목
제한 횟수 숫자 맞추기 게임
문제 설명

컴퓨터가 1부터 100 사이의 랜덤 숫자를 하나 만든다.
사용자는 최대 7번까지 숫자를 입력해서 맞힌다.

조건:

입력값이 정답보다 작으면 UP
입력값이 정답보다 크면 DOWN
맞히면 즉시 종료
7번 안에 못 맞히면 실패
범위 밖 숫자는 횟수에 포함하지 않고 다시 입력받는다.
같은 숫자를 다시 입력하면 횟수에 포함하지 않는다.

같은 숫자를 다시 입력했는지 확인하기 위해 입력 기록 배열을 사용할 수 있다.
