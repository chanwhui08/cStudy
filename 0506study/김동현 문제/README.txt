상우
- 흐름이 명확하고 초급 배열 문제로 좋음
- 입력 범위, 예외처리, 출력 형식 설명이 부족함
민철
- 완전탐색과 비교 로직 연습에 좋음
- 중복 숫자, 잘못된 입력 처리 기준이 없음
찬휘 1
- 조건 설계가 재밌고 사고력이 필요함
- 입력 범위와 출력 예시가 부족함
찬휘 2
- 구간 탐색과 시간복잡도 고민 요소가 좋음
- 문제 난이도 대비 설명과 예시가 부족함
국정이형 1
- 연속 구간 처리 흐름 연습에 좋음
- 입력 조건과 출력 형식 설명이 부족함
국정이형 2
- 조건 분기와 중복 검사 연습에 좋음
- 예외 입력과 종료 조건 설명이 부족함
